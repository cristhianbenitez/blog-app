## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

2. Clone the repo

   ```sh
   git clone https://github.com/cristhianbenitez/blog-app.git
   ```

3. Install NPM packages

   ```sh
   npm install
   ```

4. Run the application

   ```sh
   npm run start
   ```
